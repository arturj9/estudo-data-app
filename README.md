## Descrição
Uma aplicação de dados para fins de estudo do Streamlit. 

## Tecnologias (exemplos)
- Python, Pandas, NumPy
- Streamlit
- Poetry

## Pré-requisitos
- 3.12 >= Python < 3.14
- pip ou poetry

## Instalação

### Usando pip
```bash
git clone <repo-url>
cd estudo-data-app
python -m venv .venv
# Linux / macOS
source .venv/bin/activate
# Windows (PowerShell)
.venv\Scripts\Activate.ps1
# Windows (CMD)
.venv\Scripts\activate.bat
pip install -r requirements.txt
```

### Usando Poetry
```bash
git clone <repo-url>
cd estudo-data-app
poetry install
poetry env activate
<caminho_para_ativar_env>
```

## Executando
```bash
streamlit run app.py
```
